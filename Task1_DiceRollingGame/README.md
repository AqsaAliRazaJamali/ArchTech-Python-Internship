#  Dice Rolling Game

> A console-based Python application that simulates rolling a pair of six-sided dice using Python's built-in random number generator.

---

## 📖 Project Overview

The Dice Rolling Game is a beginner-friendly Python project that demonstrates the use of random number generation, loops, functions, and user interaction through a simple console interface.

Each time the program runs, two random numbers between **1** and **6** are generated to represent the outcome of rolling two dice. After displaying the results and their total, the user can choose to roll again or exit the application.

This project was completed as **Task 1** during the **Python Development Internship** at **Arch Technologies**.

---

##  Features

-  Simulates rolling two six-sided dice
-  Generates random values between **1** and **6**
-  Displays the total of both dice
-  Allows unlimited re-rolls until the user exits
-  Clean and user-friendly console interface
-  Modular code using Python functions

##  How It Works

1. The program generates two random numbers between **1** and **6**.
2. The values of both dice are displayed.
3. The total of both dice is calculated and shown.
4. The user is prompted to roll again.
5. The process repeats until the user chooses to exit.
